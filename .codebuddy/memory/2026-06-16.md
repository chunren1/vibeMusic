## 前端性能优化 (流畅度提升)

### Nginx 层 (nginx.conf)
- **Gzip**: `gzip_min_length 256` 避免小响应压缩开销；`gzip_comp_level 6→4` 减少 CPU；`gzip_types` 补全 `text/html` + `font/woff2`
- **连接池**: 新增 `upstream backend { keepalive 32 }`，所有 proxy_pass 改用 `http://backend` + `proxy_set_header Connection ""`，消除每次请求 TCP 握手
- **并发**: `worker_connections 1024→2048` + `multi_accept on`
- **Keepalive**: `keepalive_timeout 65→30` + `keepalive_requests 200`
- **缓存**: `/uploads/` 从 `expires 7d` 改为 `1d + must-revalidate`，用户换头像立即生效

### Vite 构建 (vite.config.js)
- **target**: `es2015→es2020` — 现代浏览器原生 async/await / optional chaining，bundle 缩小约 15%
- **Chunk 拆分**: `vue+pinia+@vue` 合并 → 拆为 `vue-core` / `pinia` 独立 chunk，并行下载更快
- **chunkSizeWarningLimit**: 500KB 阈值
